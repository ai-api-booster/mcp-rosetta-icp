# CallRequest

CallRequest is the input to the `/call` endpoint.

## Example Usage

```typescript
import { CallRequest } from "icpmcp";

let value: CallRequest = {
  networkIdentifier: {
    blockchain: "bitcoin",
    network: "mainnet",
    subNetworkIdentifier: {
      network: "shard 1",
      metadata: {},
    },
  },
  method: "eth_call",
  parameters: {},
};
```
