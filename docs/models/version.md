# Version

The Version object is utilized to inform the client of the versions of different components of the Rosetta implementation.

## Example Usage

```typescript
import { Version } from "icpmcp";

let value: Version = {
  rosettaVersion: "1.2.5",
  nodeVersion: "1.0.2",
  middlewareVersion: "0.2.7",
};
```
