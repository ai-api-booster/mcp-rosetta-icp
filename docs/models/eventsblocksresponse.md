# EventsBlocksResponse

EventsBlocksResponse contains an ordered collection of BlockEvents and the max retrievable sequence.

## Example Usage

```typescript
import { EventsBlocksResponse } from "icpmcp";

let value: EventsBlocksResponse = {
  maxSequence: 5,
  events: [],
};
```
