# NetworkIdentifier

The network_identifier specifies which network a particular object is associated with.

## Example Usage

```typescript
import { NetworkIdentifier } from "icpmcp";

let value: NetworkIdentifier = {
  blockchain: "bitcoin",
  network: "mainnet",
  subNetworkIdentifier: {
    network: "shard 1",
    metadata: {},
  },
};
```
